# utilitype

Provides utility functions in typescript.

## strings

### capitalize
Capitalize the given string

```typescript
capitalize('VOLVO') => 'Volvo'
```

### tokenize
```typescript
tokenize('VOLVO_V40') => ['VOLVO', 'V40']

tokenize('Audi R8') => ['Audi', 'R8']
```

## Contributing
